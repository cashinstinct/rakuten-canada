## Pourquoi cette page existe

La compétition francophone pour le mot-clé **« code parrainage Rakuten Canada »** est quasi inexistante. Volume de recherche modeste mais très ciblé (intention commerciale claire). Une page statique bien structurée peut se positionner en première page rapidement avec un effort minimal.

---

## Stack technique

- HTML5 statique pur — aucune dépendance, aucun framework
- CSS custom properties (variables) avec support dark/light mode
- Polices Google Fonts : Inter + Sora
- Schema.org JSON-LD : `Product` (avec `Review`) + `FAQPage`
- Open Graph + Twitter Card
- `hreflang` fr-CA + canonical
- Dark mode toggle JS vanilla (< 20 lignes)
